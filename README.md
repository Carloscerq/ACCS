## Como usar

A aplicação usa poetry para cuidar das dependências. Com o poetry instalado, basta rodar:

```bash
poetry install
```

Para "entrar" no shell do poetry, basta rodar:

```bash
poetry shell
```

Dentro do shell do poetry, basta rodar os comandos que deseja.
